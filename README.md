## Cách sử dụng

**Với Maven:**<br>
```maven
<repositories>
    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<dependencies>
    <dependency>
        <groupId>me.sfclog.itemlang</groupId>
        <artifactId>Main</artifactId>
        <version>1.0-SNAPSHOT</version>
    </dependency>
</dependencies>
```
